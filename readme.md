# 🏰 Tower Defense – «Королевство»

Добро пожаловать в **Tower Defense «Королевство»** — полноценную браузерную игру в жанре Tower Defense, написанную на чистом JavaScript (ES-модули) с использованием HTML5 Canvas, CSS3 и Bootstrap 5.

Игра предлагает 20+ волн врагов, 3 типа башен с улучшениями, 4 короля с уникальными способностями, систему перманентных улучшений, случайные события (руны, торговец, погода), достижения, сохранение прогресса в `localStorage` и многое другое.

---

## 🚀 Быстрый старт

### Требования
- Современный браузер (Chrome, Firefox, Edge, Safari)
- **Python 3** (для запуска локального сервера) или любой другой HTTP-сервер

### Запуск игры

1. **Скачайте или клонируйте** все файлы проекта в одну папку.
2. **Откройте терминал** (командную строку) в корневой папке проекта.
3. **Запустите локальный HTTP-сервер**:

   ```bash
   python3 -m http.server 8080
   ```

   (Если у вас Python 2, используйте `python -m SimpleHTTPServer 8080`)

4. **Откройте браузер** и перейдите по адресу:  
   ```
   http://localhost:8080
   ```

5. Наслаждайтесь игрой! 🎮

> **Важно:** игра использует ES-модули, поэтому **не открывайте** `index.html` напрямую из файловой системы (через `file://`). Всегда используйте локальный сервер.

## 🕹️ Игровой процесс

### Цель
Защитить свой замок от врагов, строя башни вдоль их маршрута. Каждая волна становится сложнее. Пройдите все 20 волн и одержите победу!

### Основные механики

#### 🗼 Башни
| Тип | Стоимость | Урон | Скорость | Радиус | Особенности |
|-----|-----------|------|----------|--------|-------------|
| **Быстрая** | 50 | 10–15 | 0.5 сек | 200 | Высокая скорострельность |
| **Обычная** | 80 | 25–35 | 1.0 сек | 250 | Сбалансированная |
| **Снайпер** | 150 | 60–80 | 2.5 сек | 400 | Игнорирует 50% брони врага |

- **Улучшение:** каждая башня может быть улучшена до 5-го уровня (урон, скорость, радиус растут).
- **Элементы:** можно добавить к башне один из трёх элементов (Огонь, Лёд, Яд) за 50 золота:
  - 🔥 Огонь – наносит дополнительный урон (горение) в течение 3 сек.
  - ❄️ Лёд – замораживает врага на 1 сек.
  - ☠️ Яд – снижает броню врага на 30% на 4 сек.

#### 👾 Враги и боссы
- Враги идут по заданному пути. При достижении конца они наносят урон замку.
- С каждой волной здоровье, скорость и количество врагов растут.
- **Мини-боссы** (20% шанс) – вдвое больше здоровья и урона.
- **Боссы** появляются каждые 5 волн – здоровье ×5, урон ×2.

#### 👑 Короли и способности
Перед игрой выберите одного из четырёх королей. Способность активируется кнопкой в нижней панели.

| Король | Способность | Эффект | Перезарядка |
|--------|-------------|--------|-------------|
| **Воин** | Ярость | +50% урона всем башням на 10 сек | 60 сек |
| **Маг** | Замедление | -40% скорости всех врагов на 8 сек | 45 сек |
| **Лучник** | Меткий выстрел | +30% радиуса всех башен на 12 сек | 50 сек |
| **Инженер** | Экономия | Следующая башня строится бесплатно | 90 сек |

#### 🌦️ Погодные эффекты
Каждые 5 волн погода меняется в зависимости от локации:
- **Дождь** – скорость атаки башен -15%
- **Туман** – радиус башен -20%
- **Снегопад** – скорость врагов -20%

#### ✨ Руны (магия поля)
С некоторым шансом после убийства врага на поле появляется руна. Кликните по ней, чтобы активировать:
- 🟢 Зелёная – ускоряет башни на 20% на 10 сек.
- 🔵 Синяя – замедляет врагов на 30% на 8 сек.
- 🔴 Красная – наносит 50% от максимального здоровья всем врагам на поле (мгновенно).

Руна исчезает через 10 секунд, если её не активировать.

#### 🧙 Торговец
После каждой волны (кроме первой) с шансом 40% появляется торговец, предлагающий временное усиление за золото (действует 1–3 волны). Примеры усилений:
- +25% урона всех башен
- +20% скорости атаки всех башен
- +15% здоровья замка (восполнение)

#### 🏅 Достижения
Система достижений с бонусным опытом:
- «Первая кровь» – убить первого врага
- «Тысяча и один» – убить 1000 врагов
- «Архитектор» – построить 50 башен
- «Неприступный» – пройти 10 волн без потери здоровья замка
- «Боссобой» – убить 10 боссов

#### 🛒 Перманентные улучшения
Магазин в главном меню, где за очки опыта можно купить постоянные бонусы (5 уровней каждое):
- **Капитализм** – +10% золота с убийств за уровень
- **Крепость** – +5% здоровья замка за уровень
- **Мастерство** – -5% стоимости башен за уровень
- **Дальнозоркость** – +5% радиуса всех башен за уровень
- **Налог** – +10 золота за завершённую волну за уровень
- **Регенерация** – замок восстанавливает 5% здоровья после волны за уровень
- **Скорость строительства** – башни строятся мгновенно (уменьшение задержки на 20% за уровень)

## 🎮 Управление

- **Выбор башни** – клик по кнопке с типом башни в нижней панели, затем клик по свободному месту на поле.
- **Клик по башне** – открывает контекстное меню с информацией, кнопками улучшения, продажи и добавления элемента.
- **Клик по руне** – активирует её эффект.
- **Кнопка «Способность»** – активирует способность выбранного короля (в нижней панели).
- **Кнопка «Пауза»** – приостанавливает игру.
- **Кнопка «×2»** – ускоряет игру в 2 раза.

## ⚙️ Настройка баланса

Все игровые параметры (стоимости, уроны, шансы, множители и т.д.) находятся в файле **`js/balance.js`**. Каждый параметр снабжён комментарием для лёгкой настройки.

Пример:
```javascript
export const BALANCE = {
    towers: { ... },
    enemies: { ... },
    kings: { ... },
    // и т.д.
};
```

Вы можете изменять эти значения без необходимости править логику игры.

## 🔊 Звуки

Звуковые эффекты реализованы через Web Audio API (без внешних файлов). Все звуки синтезируются программно. Если звук не работает, убедитесь, что браузер поддерживает Web Audio API и что звук не заблокирован (автозапуск).

## 💾 Сохранение прогресса

Игра автоматически сохраняет ваш прогресс в `localStorage` браузера. Сохраняются:
- Очки опыта
- Уровни перманентных улучшений
- Статистика (убийства, построенные башни, пройденные волны, убитые боссы)
- Разблокированные достижения
- Последний выбор короля, сложности и локации

Кнопка **«Сбросить прогресс»** в главном меню полностью очищает сохранения.

## 🧪 Технические детали

- **Игровой цикл:** `requestAnimationFrame` с фиксированной дельтой времени, ускорение ×2 достигается умножением `deltaTime`.
- **Рендеринг:** Canvas 2D для производительности и гибкости.
- **Интерфейс:** Bootstrap 5 для адаптивных модальных окон и элементов управления.
- **Модули:** ES-модули (`type="module"`) для чистой архитектуры.
